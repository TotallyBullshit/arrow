xSplitAt

Usage

xSplitAt(nums, coll)

xSplitAt...(nums, ...)

Arguments

nums
a vector of nonnegative whole numbers.
coll
a collection
...
see above.

Value

A list of two lists; the first list containing the first nums elements of coll, and the second list containing the remaining elements coll.

Description

Split a collection into elements before and upto an index, and after that index.

Corner Cases

If nums is zero then the first list in the returned value is empty. Likewise, if nums is equal or larger than the length of coll then the second return list is empty. If coll is length zero both lists are empty.
