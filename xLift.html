xLift

Usage

xLift(fn, fns)


xLift...(fn, ...)


fn1 %or% fn2


fn1 %and% fn2

Arguments

fn1
a function.
fn2
a function.
fn
a binary function.
fns
a collection functions.
...
see above.

Value

Returns a unary function of x.

Description

Compose a function with two other functions.

Details

xLift takes a function that works on some type of value, and makes that function work on functions of those values.

Two partially applied version of xLift exist; %and% & %or%. These are primarily intended for use with higher-order functions that take a predicate, allowing several predicates to be stringed together into a new predicate.
