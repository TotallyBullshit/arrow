xPermute

Usage

xPermute(coll, colls)

xPermute...(coll, ...)

Arguments

coll
a collection of whole numbers.
colls
several collections of equal length to coll.
...
see above.

Value

A list of collections.

Description

Permute several collections using indices.

Corner Cases

returns the empty list is coll is length-zero.
