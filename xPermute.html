xPermute

Usage

xPermute(nums, colls)

xPermute...(nums, ...)

Arguments

nums
a collection of whole numbers. The indices by which to permute the input collections.
colls
several collections of equal length to nums. The collections to permute.
...
see above.

Value

A list of collections.

Description

Permute several collections using indices.

Details

xPermute allows several collections to be rearranged in the same way simultaneously.

colls <- list(ids = list(15, 1, 12), ages = list(17, 29, 24), weights = list(80, 82, 76))

xPermute((order colls)

Corner Cases

returns the empty list is nums is length-zero.
