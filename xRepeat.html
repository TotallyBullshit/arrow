xRepeat

Usage

xRepeat(num, coll)


xRepeat...(num, ...)

Arguments

num
a nonnegative positive number. The number of times to repeat the collection.
coll
a collection. The collection to repeat end to end.
...
see above.

Value

A list.

Description

Create a collection by repeating another collection several times.

Corner Cases

Returns the empty list if coll is length-zero or num is zero.

Examples


# Example 1
#
# Use xRepeat to repeat a sine wave.

range <- seq(0, 2*pi, len = 600)
wave <- xRepeat(5, sin(range))

plot(unlist(wave), type = 'l')
