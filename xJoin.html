xJoin

Usage

xJoin(colls)


xJoin...(...)

Arguments

colls
a collection of collections. The collections to concatenate to each other.
...
see above.

Value

A list.

Description

Concatenate several collections into one collection.

Corner Cases

Empty collections act as a unit for concatenation; concatenating the empty list to another list returns the second list, without modification.
