xAtCol

Usage

xAtCol(num, colls)


xAtCol...(num, ...)

Arguments

num
a whole number. The number of the column to select.
colls
a collection of collections. The collection of rows to select a column from.
...
see above.

Value

A list.

Description

Select a column of values from a collection of rows.

Examples


# 1. Selecting columns in data-frame like structures
#    Arrow eshews data frames, but works well with lists of rows.
#    All

df <- xList[ list(ith, letters[ith], LETTERS[ith]), ith <- 1:26 ]

# [
#   [1, a, A]
#   [2, b, A]
#      .
#      .
#   [26, z, Z]
# ]

x_(df) $ x_AtCol(2)
