xNot

Usage

xNot(pred)

Arguments

pred
a predicate. The predicate function to return a negation of.

Value

A predicate function with the same parametres as pred.

Description

Return the logical negation of a function.

Details

xNot takes a predicate function - such as is.integer( ) - and returns the negation of that function - not.integer( ).

not.integer <- xNot(is.integer)

is.integer(1L)

True

not.integer(1L)

False

In general, if a function fn returns true xNot(fn) returns false for the same value. Similarily, if fn returns false then xNot(fn) returns true for that value. Finally (R uses three-value logic) if fn returns na for a value so does its negation.
