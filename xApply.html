xApply

Usage

xApply(fn, coll)

xApply...(fn, ...)

Arguments

fn
an function of any arity. The function to call.
coll
a collection. The arguments to pass to fn. The collection may be named or unnamed; normal R function call semantics dictate how the arguments are used by the input function.
...
see above.

Value

The return value of fn.

Description

Invoke a function with a collection of arguments.

Examples


# 1. call a variadic function with xApply

nums <- c(1, 5,2,3)

xApply(sum, nums)
