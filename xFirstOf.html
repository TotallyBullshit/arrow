xFirstOf

Usage

xFirstOf(coll)


xFirstOf...(...)

Arguments

coll
a collection. The collection to get the first element of.
...
see above.

xFirstOf

Value

The first element in coll.

Description

Return the first element of a collection.

Corner Cases

Throws an error if coll has less than one element; this is because any other corner case would violate the functions type-signature.

Examples


xFirstOf...('a', 'b', 'c')



[1] "a"

x_(letters) $ xFirstOf()
