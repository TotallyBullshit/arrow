x_

Usage

x_(val)

Arguments

val
an arbitrary value. The value to wrap in an arrow object. The methods available depend on the input type; functions and collections have the most methods available.

Value

An object of class "arrow". Internally the object is represented as a list with a single field x, but this field cannot be accessed directly. Instead, the method $ x_( ) or $ x_Identity( ) can be used to return the data stored in an arrow object.

The methods available to an arrow object depend on the type of the data it contains. All arrow objects inherit a handful of methods regardless of their type; these include xIdentity and xTap - a method that allows anonymous function to be executed on an arrow object.

Matrices, data frames, and factors have methods for converting them to collections, while normal Arrow functions are also available as methods for collections and functions.

Description

Generate an arrow object with methods available to it.

Corner Cases

The methods that can be used by $ x_( ) object varies depending on the type of val.
