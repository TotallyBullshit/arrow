xUnzipNames

Usage

xUnzipNames(coll)


xUnzipNames...(...)

Arguments

coll
a named collection. The collection to split into name, value pairs.
...
see above.

Value

A named list.

Description

Split a named collection into a list of name: value list pairs.

Details

xUnzipNames is the inverse function to xZipNames - it takes a named collection and converts it into a list of key, value list pairs. This is primarily intended for allowing functions that cannot read collection names - like map and select - to be able to operate on both a value and its key simultaneouslty.

Corner Cases

Returns list() if coll is length-zero.
