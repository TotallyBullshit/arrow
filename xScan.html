xScanl

Usage

xScanl(fn, val, coll)

xScan(fn, val, coll)

xScanl...(fn, val, ...)

xScan...(fn, val, ...)

Arguments

fn
a binary function that returns a value that fn can later take as its right argument.
val
an arbitrary value.
coll
a collection.
...
see above.

Value

A list with its initial element being coll, and containing length(coll) + 1.

Description

Successively combine a list of values into a single value using a binary function (left to right, with an initial value). Return a list containing each intermediate result, and the final result.

Corner Cases

Returns list(val) if coll is length-zero.
