xAsFunction

Usage

xAsFunction(coll)

xAsFunction...(...)

Arguments

coll
a collection
...
see above.

Value

A function that takes one or more indices.

Description

Convert a collection to a function that takes an index.

Corner Cases

Returns the empty list if coll is length-zero.

Examples


# 1. xAsFunction can be an elegant way of
#    indexing a collection.

letters_fn <- xAsFunction(letters)

xMapply(
	letters_fn,
	xSetProd...(
		1:26, 1:26)
)
