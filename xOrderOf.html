xOrderOf

Usage

xOrderOf(nums)


xOrderOf...(...)

Arguments

nums
a vector of numbers. The numbers to rank in order of size.
...
see above.

Value

A vector of numbers.

Description

Return a permutation of indices that reorders an integer vector.

Details

xOrderOf returns the indices for a collection that are required to re-order it. For example,

c(3, 1, 2)[ xOrderOf(c(3,1,2)) ]

re-arranges the collection as

c(1, 2, 3)

This is a trivial use of xOrderOf; a more common use is to re-order a collection of collections by the indices of one row or column.

Corner Cases

If nums is empty then integer(0) is returned.
