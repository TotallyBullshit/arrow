xDelay

Usage

xDelay(fn, num)

Arguments

fn
an arbitrary function. The function to slow the rate of execution of.
num
a nonnegative whole number. The number of seconds to delay execution by.

Value

A function with the same parametres as fn.

Description

Delay the evalution of a function after invokation for a set amount of time.

Details

xDelay is primarily meant for use with side-effectful functions; likely examples including functions that call web API's and functions that that display graphics.

xDelay throttles the rate at which a function can be repeatedly excecuted. This can be useful in preventing API rate limits from being exceeded or to emulate a refresh rate.

Corner Cases

If num is zero then fn is returned untouched.
