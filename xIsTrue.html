xIsTrue

Usage

xIsTrue(val)

Arguments

val
an arbitrary value.
...
see above.

Value

A true or false value.

Description

Is an value true?

Corner Cases

xIsTrue returns either true or false, to make it safe for use with if statements. If val is length zero False is returned.
