xIsEmpty

Usage

xIsEmpty(coll)


xIsEmpty...(...)

Arguments

coll
a collection. The collection to test for being length zero.
...
see above.

xIsEmpty

Value

A boolean value.

Description

Is a collection length-zero?

Examples


# 1.
# xIsEmpty is used to check for a recursive function's base case below.

map <- function (fn, coll) {

	if (xIsEmpty(coll)) {
		list()
	} else {
		xJoin...(
			fn(xFirstOf(coll)),
			map(fn, xRestOf(coll)) )
	}
}
