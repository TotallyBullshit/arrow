xMaxBy

Usage

xMaxBy(fn, coll)


xMaxBy...(fn, ...)

Arguments

fn
a unary function. The function to measure the size of an element.
coll
a non-empty collection. The collection to find the largest value of.
...
see above.

xMaxBy

Value

An element from coll.

Description

Get the largest value in a collection according to a measure function.

Corner Cases

If an empty collection is given, an error is thrown. This is because is impossible to return an element from a list that has none.

Examples

# 1. Select the largest row in a data-frame like struture
#    by an attribute.

GDP <- list(
	list('Saudi Arabia', 711050 * 10^6),
	list('Switzerland', 631050 * 10^6),
	list('Netherlands', 770067 * 10^6),
	list('Iran', 551588 * 10^6)
)

x_(GDP) $ x_MaxBy(xSecondOf)



[[1]]
[1] "Netherlands"

[[2]]
[1] 7.70067e+11





# list("Netherlands", 7.70067e+11)
