xAt

Usage

xAt(nums, coll)

xAt...(nums, ...)

Arguments

nums
a vector of whole numbers. Indices to select values in coll.
coll
a collection. The collection to subset.
...
see above.

Value

A list.

Description

Select a collection at certain indices.

Details

details

Corner Cases

xAt does not allow subscripting values out of bounds, unlike base R. If an index larger than the maximum value in coll is given an error is thrown.
