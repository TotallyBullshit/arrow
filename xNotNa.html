xNotNa

Usage

xNotNa(val)

Arguments

val
an arbitrary value.The value to test for not-na.
...
see above.

Value

A true or false value.

Description

Is an value not na?

Corner Cases

xNotNa returns either true or false, to make it safe for use with if statements. If val is length zero True is returned.
