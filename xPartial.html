xPartial

Usage

xPartial(fn, coll)


xPartial...(fn, ...)

Arguments

fn
an arbitrary function. The function to have some of its arguments fixed.
coll
a collection. The arguments to fix.
...
see above.

Value

A function of equal or lesser arity to fn.

Description

Fix several arguments of a function, producing a function of smaller arity.

Details

Partial application is a mechanism for specialising a general function to a particular purpose. For example, the very general function xMap can be easily specialised to particular tasks using xPartial:

lengths <- xPartial(xMap, list(fn = length))

sizes <- xPartial(xMap, object.size))

In this case the use of xPartial can match the argument to specialise by name, or by position. The use of xPartial over xPartial... is required in the first case, to avoid a parametre name class between the argument of xMap to fix (fn) and the function passed to xPartial (fn).

Corner Cases

Partial application also works for ellipses (eg list(... = 1)).
