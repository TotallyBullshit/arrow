xPartial

Usage

xPartial(fn, coll)

xPartial...(fn, ...)

Arguments

fn
an arbitrary function.
coll
a collection
...
see above.

Value

A function of equal or lesser arity to fn.

Description

Partially apply a function.

Corner Cases

Partial application also works for ellipses (eg list(... = 1)).
