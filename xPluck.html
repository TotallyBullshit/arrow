xPluck

Usage

xPluck(str, colls)

xPluck...(str, ...)

Arguments

str
a string. The key to select from each collection.
colls
a collections of collections. The collection of collections to select keys from.
...
see above.

Value

A list.

Description

Map over a collection of lists or pairlists, selecting fields in each element by name.

Corner Cases

Returns the empty list if colls is length-zero. If str is length-zero then the empty string "" is used to match key-names.
