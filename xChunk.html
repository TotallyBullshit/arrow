xChunk

Usage

xChunk(num, coll)

xChunk...(num, ...)

Arguments

num
a nonnegative whole number. The desired length of each group of elements.
coll
a collection. The collections to divide into groups.
...
see above.

Value

A list of lists.

Description

Divide a collection into segments of fixed length.

Corner Cases

The final list in the return value will have less than num elements if length(coll) is not evenly divisible by num. if coll is length-zero, the empty list is returned.
