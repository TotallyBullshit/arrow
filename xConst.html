xConst

Usage

xConst(val)

xK(val)

Arguments

val
an arbitrary value. The value to close over with a closure.

Value

A variadic function that returns val.

Description

Create a function that returns a particular value.

Details

xConst is the contant combinator - a function that takes an arguments and returns a function that always returns that value.

The function returned by xConst has ellipsis arguments, and ignores those arguments.
