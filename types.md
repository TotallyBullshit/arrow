
# Type Signature Notation

- the container type of a variable must always be mentioned,
including for atomic values.

## Collections

[any] or [] - a list
|any| or || - a collection
||any|| or |||| - a vector

## Functions

Variadic functions are denoted with a ...

(a -> b -> ... -> c)

The null function is

Null -> Null

(V -> V)
