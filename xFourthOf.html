xFourthOf

Usage

xFourthOf(coll)


xFourthOf...(...)

Arguments

coll
a collection. The collection to get the fourth element of.
...
see above.

Value

The fourth value in coll.

Description

Return the fourth value in a collection.

Corner Cases

Throws an error if coll has less than four element; this is because any other corner case would violate the functions type-signature.
