xAsClosure

Usage

xAsClosure(fn)

Arguments

fn
an arbitrary function. Either a primitive or non-primitive function to convert to a non-primitive function.

xAsClosure

Value

A non-primitive closure.

Description

Convert a primitive function to a closure.

Details

xAsClosure takes a primitive function and wraps it in a normal R function that passes its arguments to the underlying primitive function.

The most commonly encountered R primitive functions are the arithmetic operators. xAsClosure can convert these to normal functions.

xAsClosure('+')

function (e1, e2) fn(e1, e2)

doc

Corner Cases

xAsClosure does not work for every primitive function (for example c( )), so caution should be taken when using this function.

Examples

#    primitive operations to normal R functions.

plus <- xAsClosure('+')
plus(1, 2)
