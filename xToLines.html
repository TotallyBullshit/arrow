xToLines

Usage

xToLines(str)

Arguments

str
a length-one character vector. The string to split into lines.

xToLines

Value

A character vector.

Description

Split a string at every newline character.

Details

The input string is split at newline characters. Multiple newlines are treated the same as one newline.

Examples


# 1. Lexically sort a list.
#

info <-
"
a: a for aardvark.
z: z for zebra.
d: d for deer.
f: f for ferret.
g: g for green snake.
y: y for yapok.
"

x_(info) $ xToLines() $ x_Tap(sort)

[4] "g: g for green snake." "y: y for yapok."       "z: z for zebra."
