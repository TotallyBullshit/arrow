xMapMany

Usage

xMapMany(fn, colls)

xMapMany...(fn, ...)

Arguments

fn
a n-ary function.
colls
n collections.
...
see above.

Value

A list.

Description

Map a function across many collections simultaneously.

Corner Cases

Returns the empty list if any collection is length-zero. If the collection lengths are not equal then elements are recycled in the shorter collections.

Examples


# 1. Multiply the corresponding elements of two lists.

xMapMany...('*', sample(0:9), sample(0:9))
