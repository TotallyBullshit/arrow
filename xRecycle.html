xRecycle

Usage

xRecycle(colls)


xRecycle...(...)

Arguments

colls
a collection of collections. The collection to convert to a non-ragged form.
...
see above.

Value

A list of lists.

Description

Make every subcollection in a collection the same length by cyclically reusing elements.

Details

xRecycle takes a ragged collection - a collection of collections of varying length - and recycles elements to make the shorter collections the same length as the longest collection.

xRecycle...( list('a', 'b'), 1:3 )

list(list("a", "b", "a"), list(1L, 2L, 3L))

Corner Cases

Returns the empty list if colls is length-zero, or if a collection in colls is length-zero.
