xJuxtapose

Usage

xJuxtapose(fns)

xJuxtapose...(...)

Arguments

fns
a list or pairlist of functions.
...
see above.

Value

A variadic function that returns a list.

Description

Create a function that applies an argument to a list of underlying functions.

Details

xJuxtapose is a method of applying one value to several functions simultaneously. The function returned by xJuxtapose returns a list - on element for each input function to xJuxtapose. Each element of this list contains the result of calling an element of fns with the arguments supplied to xJuxtapose's return function.

A function to summarise a data set can be implemented with xJuxtapose:

summarise <- xJuxtapose...(min, median, max)

summarise(c(3, 1, 2, 4))

list(1, 2.5, 4)

Each function passed to xJuxtapose is invoked with the data set, and the results are collected in a list.

Corner Cases

If no functions are provided the empty list is returned.
