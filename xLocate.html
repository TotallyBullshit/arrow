xLocate

Usage

xLocate(pred, coll)


xLocate...(pred, ...)

Arguments

pred
a predicate function. The function to test each element of a collection with.
coll
a collection. The collection with elements to test.
...
see above.

Value

An length-one or length-zero whole number.

Description

Get the indices of the elements that return true for a predicate.

Corner Cases

returns integer(0) if no match is found.
