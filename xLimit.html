xLimit

Usage

xLimit(fn, num)

Arguments

fn
an arbitrary function. The function to limit the number of times it can be called.
num
a nonnegative whole number. The number of times to allow the function to be called.

Value

A function with the same parametres as fn.

Description

Create a function that can call its underlying function a limited number of times.

Details

xLimit limits the number of times a function can be called. It is primarily used for functions that makes request to online services with a rate limit. xLimit makes it easy to respect a website's rate limits without adding a lot of boilerplate to your code.

require(RCurl)

url <- "http://randomword.setgetgo.com/get.php"

random_word <- xLimit(function () toString(httpGET(url)), 3)

xMap(function (ith) random_word(), 1:3)

list("mashy\r\n", "iridectomy\r\n", "cystopyelitis\r\n")

Examples


# 1. Get random words from an API.

require(RCurl)

url <- "http://randomword.setgetgo.com/get.php"

random_word <-
	x_( function (...) toString(httpGET(url)) ) $
	x_Limit(3)
