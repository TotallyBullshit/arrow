xChop

Usage

xChop(num, coll)


xChop...(num, ...)

Arguments

num
a nonnegative whole number. The desired number of output collections to generate.
coll
a collection. The collection to chop up.
...
see above.

Value

A list of lists.

Description

Divide a collection into a fixed number of segments.

Details

xChop is used to divide a collection into several segments. This differs from xChunk in that xChop creates a fixed number of segments or arbitrary size, while xChunk creates segments of fixed size, but not a fixed number of such segments.

xChop(2, 1:5)

list(list(1, 2, 3), list(4, 5))

xChop can be used for dividing a task for parallel execution. For example, a very large collection could be chopped into four subcollections by using xChop(4, coll), before applying a function to each sublist using mclapply( ).

Corner Cases

The final sublist returned by xChop may have less elements than the other sublists, depending on whether or not the length of coll is evenly divisible by num. If coll is length-zero, the empty list is returned.

Examples


# 1. Composing linear functions
#    composing linear function 'multiplies' them,
#    so composing x := 2x and x := 3x returns the
#    'product' function x := 6x

f <- x := 2*x
g <- x := 3*x

# h = x := 3(2*x)
# h = x := 6*x

h <- f %then% g

h(2)



[1] 12



# 12
