xToWords

Usage

xToWords(str)

Arguments

str
a length-one character vector. The string to split into whitespace-seperated words.

Value

A character vector, with one or more elements.

Description

Split a string at whotespaces.

Details

The input string is split at spaces, tabs, and newlines, or a mix of the above. Multiple whitespaces are treated the same as one whitespace.
