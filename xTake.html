xTake

Usage

xTake(num, coll)


xTake...(num, ...)

Arguments

num
a nonnegative whole number. The number of elements to return from the input collection.
coll
a collection. The collection to subset.
...
see above.

Value

A list.

Description

Take several elements from the head of a collection.

Corner Cases

If coll is empty the empty list is returned.
