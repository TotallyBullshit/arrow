xImplode

Usage

xImplode(str, strs)


xImplode...(str, ...)

Arguments

str
a length one character vector. The string to use as a delimiter.
strs
a collection of length one character vectors. The strings to concatenate.
...
see above.

xImplode

Value

A length-one character vector.

Description

Concatenate a character vector into a single string using a delimiter.

Examples


# 1. create a simple csv writer.

to_csv <- xPartial(xImplode, ', ')
to_csv(xTake(10, letters))



[1] "a, b, c, d, e, f, g, h, i, j"




# c('a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j')
