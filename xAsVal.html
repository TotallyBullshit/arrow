xAsVal

Usage

xAsVal(sym)

Arguments

sym
a symbol or string. The name of the variable to be converted to a value.

xAsVal

Value

Null; used for side-effect.

Description

Convert a normal R variable to a constant value.

Details

xAsVal takes a variable that exists in the calling environment, and locks it. This prevents further modification, and if any attempt is made to modify the variable an error is thrown.

Corner Cases

xAsVal throws an error if an attempt is made to convert a variable that doesn't exist in the parent frame.

Examples


# 1. xAsVal can be used to 'lock' a list of constant variables
#    so it can't accidentally be updated.

consts <- list(
    pi = 3.14,
    golden = 1.618
)

xAsVal(consts)

# consts$pi <- 2
# would throw an error.
