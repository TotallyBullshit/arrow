$ xTranspose

Usage

x_(  ) $ xTranspose()

Value

An arrow object containing a matrix.

Description

Transpose a matrix.

Details

xTranspose( ) is very similar to the base function t( ). It takes each row of an input matrix and uses it as a column in the output matrix. The dimensions of the input matrix is flipped; if a 4 x 2 matrix is supplied to xTranspose( ) then a 2 x 4 matrix is returned.

xTranspose is an involution - a function that is its own inverse. For any collection, xTranspose(xTranspose(coll)) is the original collection.

Corner Cases

The empty matrix is returned if the input matrix is empty. Matrices with zero rows or columns are transposed as expected.
