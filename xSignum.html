Usage

xSignum(nums)

xSignum...(...)

Arguments

nums
a vector of non-complex numbers.
...
see above.

Value

A integer vector of elements in the set {-1, 0, +1}.

Description

Get the sign of a vector of numbers.

Corner Cases

If nums is lenth-zero then the unit of that vector is returned. The sign of zero is zero.
