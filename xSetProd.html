xSetProd

Usage

xSetProd(colls)

xSetProd...(...)

Arguments

colls
a collection of collections. The collections to use to take the cartesian product.
...
see above.

Value

A list of collections, with as many elements per collection as there are collections in colls.

Description

Get the cartesian product of several collections.

Corner Cases

Returns the empty list if colls is length-zero.

Examples


# 1. xSetProd with xMap, xMapply or xReduce are
#    generally very nice replacements for loops.

xMapply(
    (ith : jth) := {
        ith^2 + jth^2
    },
    xSetProd...(1:4, 1:4)
)
