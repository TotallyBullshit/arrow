xCompose

Usage

xCompose(fns)


xCompose...(...)


fn1 %of% fn2


fn1 %then% fn2

Arguments

fns
a collection of functions. The functions to compose with each other.
...
see above.

Value

A variadic function.

Description

Take several functions and return a composite function.

Details

Arrow's function composition is largely done with the infix operators %of% and %then%.

%then% is the classic function composition operator;

(as.numeric %then% sqrt %then% print)('10')

Like methods, the order of function execution is from left to right; convert a string to a number then take the square root then print the value. The reversed form %of% is equally as ubiquitous.

The classic example of function composition (apart from Church numerals) is composing linear functions.

f <- x := 2*x

g <- x := 4*x

h <- x := 1*x

(f %of% g %of% h)(1)

(x := 2*(4*(1*x)) )(1)

xCompose

In this case the output of one function is piped to another:

1 => 1*1 => 1*4 => 4*2 => 8

In this case function composition behaves exactly like multiplication.

In more typical use of xCompose is to reduce the number of anonymous functions needed by some programs.

xMap(sqrt %of% sqrt, 1:10)
