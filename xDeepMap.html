xDeepMap

Usage

xDeepMap(fn, coll)


xDeepMap...(fn, ...)

Arguments

fn
a unary function. A function to recursively apply into a collection.
coll
a collection. The collection to be mapped into.
...
see above.

Value

A list or pairlist.

Description

Recursively map a function into a nested collection, preserving its structure.
