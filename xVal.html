xVal

Usage

xVal(sym, val)

Arguments

sym
a symbol or string. The variable name to assign a value.
val
an arbitrary value. The value to be assigned.

Value

Null; this function is used for side-effects.

Description

Assign a constant value to the calling environment.

Details

xVal works like normal R assignment, with the exception that if any attempt to update the assigned variable will result in an error. This is analogous to const in other languages.

the_letter_a <- "a" xVal(the_letter_a, "a")

As the above call shows xVal is a standard function, not an infix function like normal assignment.

Corner Cases

Overwrites the value referenced by symr if the variable already exists in the parent frame.
