xUnionOf `` Get the set union of several collections.

Usage

xUnionOf(colls)


xUnionOf...(...)

Arguments

colls
a collection of collections. The collections to take the union of.
...
see above.

Value

A list.

Description

xUnionOf `` Get the set union of several collections.

Details

xUnionOf returns a list of the unique elements in the combination of each collection in colls.

Corner Cases

Returns the empty list if colls is length-zero.
