xExists

Usage

xExists(pred, colls)

xExists...(pred, ...)

Arguments

pred
an predicate function that takes as many arguments as there are collections in colls.
colls
a collection of collections.
...
see above.

Value

A boolean value.

Description

Test if any ordered group of arguments from several collections is true for a predicate.

Details

xExists can be used to test if any element of a collection is true (like the base function all( )) by simply using it with the identity function.

coll <- c(True, False, True, True)

xExists...(xIdentity, coll)

When supplied with a collection of multiple collections the set product of those collections is checked with the predicate. In order for xExists to return true at least one element of the set product must return true for a predicate.

commutes <- (a : b) := { a + b == b + a }

xExists...(commutes, 1:2, 1:2)

The set product is

list(list(1, 1), list(1, 2), list(2, 1), list(2, 2))

Each member of the set product is tested with the commutes function

list(commutes(1, 1), commutes(1, 2), commutes(2, 1), commutes(2, 2))

list(True, True, True, True)

True

Corner Cases

If any collection is length zero (or no collections are given), then logical(0) is returned.

Examples


# 1. Determine if a string is a cyclic permutation of another string.
#    check if there exists a way of cyclically permutting one string
#    so that the other string is identical.

is_cyclic_permutation <- (string1 : string2) := {

    is_same_length <- nchar(string1) == nchar(string2)

    string1 <- xToChars(string1)
    string2 <- xToChars(string2)

    if (length(string1) != length(string2)) {
        False
    } else {

        x_(seq_along(string1)) $
        x_Exists...(
            ith := {
                # cyclic indices. modulo sucks in 1-indexed languages.

                indices <- ((seq_along(string1) + ith) %% length(string1)) + 1
                all(string1[indices] == string2)

            }
        )

    }

}
