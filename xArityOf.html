xArityOf

Usage

xArityOf(fn)

Arguments

fn
an function. The function to have its arity checked.

xArityOf

Value

A positive whole number.

Description

Return the number of arguments a function can accept.

Details

xArityOf returns the arity of a function - the number of arguments that can be passed to the function. R functions can be nullary (zero-arguments), have infinitely many arguments (variadic functions), or have a fixed number of arguments (non-variadic functions).

Corner Cases

If fn has an ellipsis (...) parametre then +Inf is returned, as the function can accept an infinite number of arguments.

Examples


# 1. xArityOf works on primitive functions.

xArityOf('+')



[1] 2




# 2

# 2. xArityOf also works on normal functions
#    and variadic functions.

xArityOf(Reduce)



[1] 5




# 5

xArityOf(function (a, ...) {})



[1] Inf




# Inf

# 3. xArityOf can be used to check that a function
#    given to a higher-order function is safe to
#    use.

safeMap <- function (fn, coll) {

    if (!xArityOf(fn) == 1) {
        stop("the function given to safeMap is not unary.")
    } else {
        lapply(fn, coll)
    }
}
