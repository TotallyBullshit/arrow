xExplode

Usage

xExplode(rexp, str)


xExplode...(rexp, ...)

Arguments

rexp
a regular expression. The pattern at which to split str.
str
a string. The string to split.
...
see above

xExplode

Value

A character vector.

Description

Split a string into a character vector using a regular expression.

Corner Cases

Returns the empty list if coll is length-zero. If no match is found the original string is returned. Every element of the returned character vector has one or more characters - no zero-character elements are ever generated.

Examples


# 1. Create a simple csv parser.

from_csv <- xPartial(xExplode, '[ 	]*,[ 	]*')
from_csv("field1, field2, field3")



[1] "field1" "field2" "field3"




# c("field1, field2, field3")
