xFlatten

Usage

xFlatten(num, coll)


xFlatten...(num, ...)

Arguments

num
a nonnegative whole-number. The maximum depth of collection in the return value.
coll
a list or pairlist.
...
see above.

Value

A list.

Description

Unnest a nested list to a given depth.

Details

xFlatten flattens nested collections to a specific depth. num specifies the maximum allowed depth in the returned list. When flattening a (infinitely nested) collection, num equals 1 returns a collection, num equals 2 returns a collection of collections, and so on.

Corner Cases

Returns the empty list if coll is length-zero. Flattening to infinite doesn't affect the depth of the output list. Flattening to one level flattens the list fully.
