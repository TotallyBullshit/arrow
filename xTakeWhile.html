xTakeWhile

Usage

xTakeWhile(pred, coll)

xTakeWhile...(pred, ...)

Arguments

pred
a predicate. The function to test each element of the collection with.
coll
a collection. The collection to drop elements from.
...
see above.

Value

A list.

Description

Take every element in a collection from the start until a predicate returns false.

Corner Cases

Returns the empty list if coll is length-zero or the first element of coll returns false for the predicate. Na values are considered false.
