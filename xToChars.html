xToChars

Usage

xToChars(str)

Arguments

str
a length-one character vector. The string to split into characters.

Value

A character vector of length nchar(str), with each element being a single character.

Description

Split a single string into a vector of characters.

Examples


# 1. CE: Sum of Digits
# sum the digits of a positive integer

x_(toString(12358134L)) $
xToChars() $
xMap(as.numeric) $
x_Reduce("+")
